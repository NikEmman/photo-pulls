# Photo-pulls

## Description
Photo-pulls is a rails app that works with Flickr API. At the home page, the user submits a flickr user_id in the field, and the app displays the id owner's public pictures.

## Project details
This is a project for [this](https://www.theodinproject.com/lessons/ruby-on-rails-flickr-api) lesson of [TheOdinProject](https://www.theodinproject.com/)

## How to use
1. Open your terminal and type `git clone git@github.com:NikEmman/photo-pulls.git`
1. Navigate into the folder by `cd photo-pulls`
1. Run the command `bundle` to install dependencies
1. Run the coummand `rails s` to start local server
1. On your browser visit [this url](localhost:3000/)
